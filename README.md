# SHAPE
A Python-based visualization tool for generating 3D nuclear shapes using quadrupole deformation parameters. This script provides an intuitive way to explore how nuclear shapes evolve with deformation ( 𝛽 β) and triaxiality ( 𝛾 γ). By Manfred Jaftha and Andile Zulu
## Requirements

The following Python libraries are required:

`numpy`
`matplotlib`

## Usage

```
python SHAPE.py
```

The user will be prompted to enter the following values: 
- Z (atomic number)
- A (atomic mass number)
- Qs (spectroscopic quadrupole)
- gamma (𝛾 deformation parameter)

A prompt to display a grid and axis is also given.
